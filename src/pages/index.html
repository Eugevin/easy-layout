Thanks for using this bundler! First of all your need to delete files in ./src/assets/* folders

With npm run dev you can start develop your layout with inline-source-maps, hot reload and other cool features for develop

With npm run build you can compress and optimize your code for production

Active packages: bootstrap.grid.min.css, normalize.css, babel, autoprefixer

All pages stored in ./src/pages/. If you want to create new .html file, you need do the restart of webpack compiler!
